This project has been develop using Python  3.7 so should be ran using this version for the most reliable experience.
To run the program please run the launcher.py file.
The program realise on a number of third party modules. A virtual environment with the correct dependencies has
be provided in the venv directory. Alternatively they can be installed manually. The versions used in development is provided bellow:
pyglet 1.4.8
pymunk 5.6.0
